# 亮度细调（明るさ微調整）

1% 刻みでの明るさ調整に対応した、macOS 向けの明るさ微調整ツールです。

## ダウンロード

Releases から `fine-brightness.dmg` をダウンロードしてください。

## インストール

1. DMG を開きます。
2. `亮度细调.app`（明るさ微調整）を Applications にドラッグします。

初回起動時に、開発元を確認できないと macOS に表示された場合は、App を右クリックして「開く」を選択してください。
