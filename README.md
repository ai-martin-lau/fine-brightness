# 亮度细调

macOS 亮度细调工具，支持 1% 亮度增减。

## 下载

请到 Releases 下载 `fine-brightness.dmg`。

## 安装

1. 打开 DMG。
2. 把 `亮度细调.app` 拖到 Applications。

首次打开时，如果 macOS 提示无法验证开发者，请右键 App 选择“打开”。
