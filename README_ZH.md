# Fine Brightness

Fine Brightness 是一个 macOS 亮度细调工具，支持 1% 亮度增减。

## 下载

下载最新版本：[`fine-brightness.dmg`](https://github.com/ai-martin-lau/fine-brightness/releases/latest/download/fine-brightness.dmg)

## 安装

1. 打开 DMG。
2. 把 `Fine Brightness.app` 拖到 Applications。

首次打开时，如果 macOS 提示无法验证开发者，请右键 App 选择“打开”。
