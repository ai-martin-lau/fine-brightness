# 亮度细调（밝기 미세 조정）

1% 단위로 밝기를 조정할 수 있는 macOS용 밝기 미세 조정 도구입니다.

## 다운로드

Releases에서 `fine-brightness.dmg`를 다운로드하세요.

## 설치

1. DMG를 엽니다.
2. `亮度细调.app`(밝기 미세 조정)를 Applications로 드래그합니다.

처음 실행할 때 macOS에서 개발자를 확인할 수 없다는 메시지가 표시되면, App을 마우스 오른쪽 버튼으로 클릭한 뒤 "열기"를 선택하세요.
